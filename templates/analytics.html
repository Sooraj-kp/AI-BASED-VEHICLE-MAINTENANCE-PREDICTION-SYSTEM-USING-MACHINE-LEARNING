The model builds random isolation trees. Anomalous engine readings are isolated in fewer splits (shorter path length) than normal readings — giving a 0–100 anomaly score. Score >50 flags the engine as anomalous. Trained only on normal data so it generalises to any type of previously-unseen fault pattern.
